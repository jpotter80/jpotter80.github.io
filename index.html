ABOUT ME

Hi, I'm James Potter, a data developer based in the scenic mountains of Black Mountain, North Carolina. My journey into data analytics began with a foundation in bioscience and a passion for uncovering insights from complex datasets.

What I do

I specialize in transforming raw data into actionable insights through analytics and visualization. Recently certified in Google Data Analytics, I work with Python, R, SQL, and various visualization tools to bring data stories to life. I'm particularly interested in creating interactive dashboards that make data accessible and meaningful.

Recent Projects

I recently deployed my first Observable Framework project - a real-time weather dashboard for Western North Carolina. This project showcases my skills in: interactive data visualization, CI/CD pipeline implementation, cloud deployment, and real-time data processing.
